# Faith Feed

A modern, mobile-responsive faith-based social media application built with Next.js, Tailwind CSS, and Lucide-react icons.

## Features

- **Sticky Header**: Navigation with search and profile avatar
- **Daily Verse**: Featured scripture card with gradient background
- **Faith Feed**: Scrollable posts with author info, timestamps, and content
- **Interactive Actions**: Like, comment, and share buttons
- **Input Bar**: Fixed bottom input for sharing reflections
- **Mobile Navigation**: Bottom tab bar with Home, Discover, Bookmarks, Community
- **Responsive Design**: Mobile-first with clean SaaS aesthetic

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Design Features

- Clean, modern UI with soft shadows and rounded corners
- Serene color palette with blues and teals
- Mobile-optimized layout
- Interactive like functionality
- Gradient backgrounds for featured content

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** for styling
